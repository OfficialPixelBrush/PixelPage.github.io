Controls

(Note: a US QWERTY Keyboard is assumed by default, so if you're using a different kind of keyboard, some keys may be mapped wrong!)
Q - color left
E - color right
R - increase brush size
F - decrease brush size
1 - save ppm
2 - load ppm
Left Click - Draw
Right Click - Color Picker

Example Files

(Note: Images may appear stretched due to being formated to 4:3, as they would be if displayed on a CRT Monitor)
C:\PIX.PPM - A draw-over of mine
C:\HELLO.PPM - A redraw of the Apple "hello."
C:\SPECT.PPM - A color Spectrum that shows most, if not all, colors this pallette has
C:\CALIB.PPM - A stock TV Calibration image
