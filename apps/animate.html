Controls

(Note: a US QWERTY Keyboard is assumed by default, so if you're using a different kind of keyboard, some keys may be mapped wrong!)
Q - frame back
E - frame forward
T - color left
G - color right
R - increase brush size
F - decrease brush size
1 - save apm
2 - load apm
Left Click - Draw
Right Click - Color Picker
